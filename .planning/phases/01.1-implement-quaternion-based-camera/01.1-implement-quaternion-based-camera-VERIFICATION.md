### Implementation Details

**Camera State:**
- Position: `vec3` (stores X, Y, Z coordinates)
- Orientation: `quat` (stores rotation, eliminates gimbal lock)
- Distance: `number` (legacy field, not currently used in shader calculations)

**Rotation Algorithm:**
```
1. Calculate pitch change from mouse Y delta
2. Calculate yaw change from mouse X delta
3. Create temporary quaternion
4. Apply pitch rotation: quat.rotateX(temp, orientation, pitchChange)
5. Apply yaw rotation: quat.rotateY(orientation, temp, yawChange)
6. Normalize: quat.normalize(orientation, orientation) // Prevents drift
```

**Movement Algorithm:**
```
1. Get forward vector: vec3.transformQuat([0, 0, -1], orientation)
2. Get right vector: vec3.transformQuat([1, 0, 0], orientation)
3. Get up vector: vec3.transformQuat([0, 1, 0], orientation)
4. Add/subtract scaled vectors to position based on input
```

**View Matrix Construction:**
```
1. Extract forward and up vectors from quaternion (camera's local space)
2. Calculate target: position + forward
3. Create view matrix: mat4.lookAt(view, position, target, up)
4. Create projection matrix: mat4.perspective(fov, aspect, near, far)
5. Combine: mvp = projection * view
```
