       OLD: return new Vec3(...)
       NEW: const v = vec3.create(); vec3.set(v, ...); return v

       NEW: const up = vec3.create()
            vec3.cross(up, this.getRight(), this.getForward())

    Key gl-matrix API differences:
    - vec3.add(out, a, b): out = a + b (requires output parameter)
    - vec3.sub(out, a, b): out = a - b
    - vec3.scale(out, a, s): out = a * s
    - vec3.cross(out, a, b): out = a × b
