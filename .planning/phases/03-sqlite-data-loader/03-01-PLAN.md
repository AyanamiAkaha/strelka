    1. Add TableInfo and SqliteQueryResult interfaces to src/core/types.ts:
       ```typescript
       export interface TableInfo {
         name: string
         hasCluster: boolean
       }

       export interface SqliteQueryResult {
         columns: string[]
         values: unknown[][]
       }
       ```

    2. Create validateTableSchema() in src/core/validators.ts:
       - Accept db (Database instance) and tableName (string) parameters
       - Query PRAGMA table_info(tableName) to get schema
       - Extract column names from result (column index 1: 'name')
       - Check for required columns: ['x', 'y', 'z']
       - Build missingColumns array of required columns not found
       - Check if 'cluster' column exists for hasCluster boolean
       - If missingColumns.length > 0: throw Error(`Table must have x, y, z columns. Table ${tableName} missing: ${missingColumns.join(', ')}`)
       - If no results from PRAGMA: throw Error(`Table not found: ${tableName}`)
       - Return hasCluster boolean

    Pattern from RESEARCH.md: Use PRAGMA table_info for schema validation, exact case-sensitive column name comparison.

    Files modified:

    - src/core/validators.ts: Add validateTableSchema() export
  </action>
